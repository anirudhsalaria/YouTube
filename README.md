### YouTube
This is a YouTube-Clone, made using ReactJs. Here you can simply search your favourite youtube videos and watch them.                                           
### Stack used
1. ReactJs
2. Material UI
3. Rapid API

Website Link - https://goyoutube.netlify.app/
